# PoE Kingsmarch Dust Calculator

An AutoHotkey v2 script for Path of Exile that calculates the Disenchant (Dust) value of unique items.

## Features
- Scrapes the latest data from PoeDB.
- Calculates dust values based on item level scaling.
- Factors in Quality, Influence, and Corrupted Implicits.

## How to use
1. Install [AutoHotkey v2](https://www.autohotkey.com/).
2. Run `dust_calculator.ahk`.
3. In-game, hover over a Unique item and press **F4**.
4. Left-click or Right-click to dismiss the overlay.
