# PoE Kingsmarch Dust Calculator

An AutoHotkey v2 script for Path of Exile that calculates the Dust value of unique items.

## Features
- Scrapes the dust data from [PoEDB](https://poedb.tw/us/Kingsmarch#Disenchant)
- Calculates dust values based on item level scaling.
- Factors in Quality, Influence, and Corrupted Implicits.

## How to use
1. Install [AutoHotkey v2](https://www.autohotkey.com/).
2. Run `dust_calculator.ahk`.
3. In-game, hover over a unique item and press **F4** to display the dust value overlay.
4. You can Left-click or Right-click to close the overlay.
